# mafiastarlite_frontend

This is the frontend for MafiAStarLite. Written in Vue.js 3 and Tailwindcss.

## Set up the project with
```
npm install
```

### Use for Dev:
```
npm run serve
```

### Build for Production:
```
npm run build
```
